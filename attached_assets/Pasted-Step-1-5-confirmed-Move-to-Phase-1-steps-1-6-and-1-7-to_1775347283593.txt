SQL for pipeline cards:
sqlSELECT
  COUNT(*) FILTER (WHERE status = 'paid') as paid_count,
  COALESCE(SUM(amount) FILTER (WHERE status = 'paid'), 0) as paid_amount,
  COUNT(*) FILTER (WHERE status IN ('submitted','acknowledged','pending')) as in_process_count,
  COALESCE(SUM(amount) FILTER (WHERE status IN ('submitted','acknowledged','pending')), 0) as in_process_amount,
  COUNT(*) FILTER (WHERE status = 'draft') as draft_count,
  COALESCE(SUM(amount) FILTER (WHERE status = 'draft'), 0) as draft_amount,
  COUNT(*) FILTER (WHERE status IN ('denied','suspended')) as denied_count,
  COALESCE(SUM(amount) FILTER (WHERE status IN ('denied','suspended')), 0) as denied_amount
FROM claims
SQL for stale drafts alert:
sqlSELECT COUNT(*) FROM claims WHERE status = 'draft' AND created_at < NOW() - INTERVAL '7 days'
SQL for recent claims with patient name:
sqlSELECT c.*, 
  COALESCE(p.first_name || ' ' || p.last_name, p.lead_id, 'Unknown Patient') as patient_name
FROM claims c
LEFT JOIN patients p ON c.patient_id = p.id
ORDER BY c.created_at DESC
LIMIT 10

Compliance Reports (/billing/intelligence/reports) — admin only:
Four report cards, each with a description, date range picker, and "Generate PDF" button. Use @react-pdf/renderer (already installed). Generate client-side.
Report 1 — Access Report:
sqlSELECT al.*, u.email as user_email 
FROM activity_logs al
LEFT JOIN users u ON al.performed_by = u.id
WHERE al.activity_type IN ('view_patient','view_claim','exported')
AND al.created_at BETWEEN $1 AND $2
ORDER BY al.created_at DESC
Report 2 — Edit History:
sqlSELECT al.*, u.email as user_email
FROM activity_logs al  
LEFT JOIN users u ON al.performed_by = u.id
WHERE al.field IS NOT NULL
AND al.created_at BETWEEN $1 AND $2
ORDER BY al.created_at DESC
Report 3 — Export Report:
sqlSELECT al.*, u.email as user_email, c.amount, c.status
FROM activity_logs al
LEFT JOIN users u ON al.performed_by = u.id
LEFT JOIN claims c ON al.claim_id = c.id
WHERE al.activity_type = 'export_pdf'
AND al.created_at BETWEEN $1 AND $2
ORDER BY al.created_at DESC
Report 4 — Claims Integrity:
sqlSELECT c.id, c.status, c.amount, c.created_at, c.updated_at, c.service_date,
  c.readiness_status, c.submission_method,
  COALESCE(p.first_name || ' ' || p.last_name, 'Unknown') as patient_name
FROM claims c
LEFT JOIN patients p ON c.patient_id = p.id
WHERE c.created_at BETWEEN $1 AND $2
ORDER BY c.created_at DESC
