One thing to watch while it builds: the NPI Luhn validation. This is a specific algorithm that some AI tools get wrong by implementing plain Luhn instead of the NPI variant which prepends 80840 before running the checksum. If the validation rejects valid NPIs or accepts invalid ones, that's the cause. The exact function is in Section 7.1 of the prompt — make sure it uses the 80840 prefix.
